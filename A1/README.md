## 1. Web Application: ZenTasks

A modern, minimalist task management application featuring a glassmorphism design and Kanban board using React and TypeScript.

- **Path**: `zen-tasks/`
- **Tech Stack**: React, TypeScript, Vite, Tailwind CSS v4, Framer Motion.
- **Features**: Drag-and-drop Kanban board, glassmorphic UI, responsive design.
- **Documentation**: [README.md](zen-tasks/README.md) | [Walkthrough](zen-tasks/walkthrough.md)
- **Deployed**: [ZenTasks](https://sjsu-a2-task1-ag-db.web.app/)

## 2. Mobile Application: Water Tracker

A cross-platform Flutter application for tracking daily water intake with local notifications.

- **Path**: `water_tracker/`
- **Tech Stack**: Flutter (Dart), Provider.
- **Features**: Daily tracking, goal setting, local notifications, dark mode layout.
- **Documentation**: [README.md](water_tracker/README.md) | [Walkthrough](water_tracker/walkthrough.md)
- **Deployed**: [Water Tracker](https://parth0248.github.io/CMPE-258-MachineLearning/)

## 3. Neural Network Classifier

A Neural Network implemented in Keras/TensorFlow to classify handwritten digits from the MNIST dataset.

- **Path**: `NeuralNetwork/`
- **Model**: Feed-forward NN with two hidden layers and dropout.
- **Performance**: Achieved ~97.5% test accuracy.
- **Artifacts**: Training history plot, Confusion Matrix.
- **Documentation**: [README.md](NeuralNetwork/README.md) | [Walkthrough](NeuralNetwork/walkthrough.md)

## 4. Colab: Image Generation & Understanding

A Jupyter Notebook demonstrating generative AI capabilities using Gemini Pro Vision (or equivalent models).

- **File**: `ImageGen_ImageUnd.ipynb`
- **Tasks**:
    1.  **Image Generation**: Generating creative images from text prompts (e.g., "A crystalline castle floating in a nebula").
    2.  **Image Understanding**: Analyzing generated images to describe lighting, architecture, etc.
- **Outputs**: Embedded directly in the notebook.
