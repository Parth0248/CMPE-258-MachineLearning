# ZenTasks

ZenTasks is a modern, minimalist task management application featuring a glassmorphism design, smooth animations, and drag-and-drop Kanban functionality.

## Features

- **Modern UI**: Built with Tailwind CSS v4 and Framer Motion.
- **Glassmorphism**: Premium aesthetic with backdrop blur and gradients.
- **Kanban Board**: Drag and drop tasks between Todo, In Progress, and Done columns.
- **Interactive**: Floating action button for quick task creation.
- **Frontend**: React + TypeScript + Vite.
- **Backend (Optional for Demo)**: Python (FastAPI) + Firebase.

## Getting Started

1. **Install Dependencies**:
   ```bash
   cd zen-tasks/web
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build fo Production**:
   ```bash
   npm run build
   npm run preview
   ```

## Deploy

This project is configured for Firebase Hosting.
1. `firebase login`
2. `firebase init`
3. `firebase deploy`
